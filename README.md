# ⚖️ Kalkulator Faraid Sederhana (Python)

Program Python sederhana untuk membantu menghitung pembagian harta waris berdasarkan hukum Islam (Faraid). Project ini dibuat sebagai alat bantu edukasi, khususnya bagi mahasiswa Hukum Keluarga Islam (HKI). ⚖️🐍

## ✨ Fitur
* 💰 Menghitung total harta warisan.
* 👥 Mendukung perhitungan untuk Suami/Istri (dengan/tanpa anak).
* 👦👧 Pembagian Ashabah untuk anak laki-laki dan perempuan (Rasio 2:1).
* 🤖 Interface terminal yang interaktif dan mudah dipahami.

## 🛠️ Cara Penggunaan (Guide) for pc

Pastiin lo udah install Python di komputer lo. Kalau belum, download di [python.org](https://www.python.org/).

1. Clone repositori ini atau download file .py-nya.
2. Buka Terminal atau CMD.
3. Masuk ke folder tempat lo nyimpen filenya.
4. Jalankan perintah:
   `bash
   python "Calculator Waris.py"

## 🛠️ Cara Penggunaan (Guide) For Android

1. Download Termux Di Playstore
2. Masukan comand ini di termux: pkg update && pkg upgrade
3. pkg install python
4. pkg install git
5. clone repository nya
6. ls
7. masuk ke folder tools nya
8. Run scriptnya
